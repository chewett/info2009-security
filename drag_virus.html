Definition

  • A destructive program that masquerades as a benign application.
  • Malicious software that is attached to another program for the purpose of executing a particular unwanted function.
  • Executes code and installs copies of itself in the memory of the infected computer, which can, in turn, infect other hosts.
  • Malicious software designed to track and spy on your activities in order to gain confidential information.
  • Malicious software which logs all key strokes made on your computer. Used to find passwords.
